= Aggiornare i prodotti di terze parti
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
È possibile aggiornare prodotti di terze parti, come JRE, su Unified Manager quando installato su sistemi Windows.

Le aziende che sviluppano questi prodotti di terze parti segnalano regolarmente vulnerabilità di sicurezza.  Puoi effettuare l'aggiornamento alle versioni più recenti di questo software secondo i tuoi tempi.
