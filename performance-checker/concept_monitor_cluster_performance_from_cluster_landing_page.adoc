= Monitora le prestazioni del cluster dalla pagina di destinazione del cluster delle prestazioni
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
La pagina di destinazione del cluster delle prestazioni visualizza lo stato delle prestazioni di alto livello di un cluster selezionato monitorato da un'istanza di Unified Manager.  Questa pagina consente di valutare le prestazioni complessive di un cluster specifico e di annotare, individuare o assegnare rapidamente per la risoluzione tutti gli eventi specifici del cluster identificati.
