= Informazioni sulle quote
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Le quote consentono di limitare o tenere traccia dello spazio su disco e del numero di file utilizzati da un utente, un gruppo o un qtree.  Si specificano le quote utilizzando `/etc/quotas` file.  Le quote vengono applicate a un volume o qtree specifico.
