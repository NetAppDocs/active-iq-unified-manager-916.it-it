= Scarica il file di aggiornamento di Unified Manager
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Prima di aggiornare Unified Manager, scaricare il file di aggiornamento di Unified Manager dal sito di supporto NetApp .

.Prima di iniziare
Dovresti avere le credenziali di accesso al sito di supporto NetApp .

.Passi
. Accedi al sito di supporto NetApp :
+
https://mysupport.netapp.com/site/products/all/details/activeiq-unified-manager/downloads-tab["Sito di supporto NetApp"]

. Vai alla pagina Download per aggiornare Unified Manager su VMware vSphere.
. Scarica il `.iso` immagine per l'aggiornamento e salvarla in una directory locale o di rete accessibile al tuo vSphere Client.
. Verificare il checksum per assicurarsi che il software sia stato scaricato correttamente.
