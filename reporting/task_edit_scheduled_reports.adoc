= Modifica i report pianificati
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Dopo aver pianificato i report, è possibile modificarli nella pagina Pianificazioni report.

.Prima di iniziare
* È necessario disporre del ruolo di Amministratore dell'applicazione o Amministratore dell'archiviazione.


.Passi
. Nel riquadro di navigazione a sinistra, fare clic su *Gestione archiviazione* > *Pianificazioni report*.
+
image::../media/scheduled_reports_2.gif[Uno screenshot dell'interfaccia utente che mostra i report pianificati.]

+
[NOTE]
====
Se si dispone delle autorizzazioni appropriate, è possibile modificare qualsiasi report e la relativa pianificazione nel sistema.

====
. Clicca sull'icona "Altro"image:../media/more_icon.gif[""] per l'orario che vuoi modificare.
. Fare clic su *Modifica*.
. È possibile modificare il *Nome pianificazione*, l'elenco *Destinatari*, la *Frequenza* e il *Formato* per la pianificazione del report.
. Al termine, fai clic sul segno di spunta per salvare le modifiche.
