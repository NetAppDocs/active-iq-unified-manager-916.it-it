= Comprendere l'interfaccia utente
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
L'interfaccia utente di Unified Manager è costituita principalmente da una dashboard che fornisce una panoramica immediata degli oggetti monitorati.  L'interfaccia utente consente inoltre di visualizzare tutti gli oggetti del cluster.

È possibile selezionare la visualizzazione preferita e utilizzare i pulsanti di azione secondo necessità.  La configurazione dello schermo viene salvata in un'area di lavoro, in modo che tutte le funzionalità necessarie siano disponibili all'avvio di Unified Manager.  Tuttavia, quando si passa da una vista all'altra e poi si torna indietro, la vista potrebbe non essere la stessa.
