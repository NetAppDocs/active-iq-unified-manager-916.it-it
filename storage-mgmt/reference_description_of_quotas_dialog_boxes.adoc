= Descrizione delle finestre di dialogo delle quote
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
È possibile utilizzare l'opzione appropriata nella scheda Quote utente e gruppo nella vista Integrità: tutte le VM di archiviazione per configurare il formato della notifica e-mail inviata quando si verifica un problema relativo alla quota e per configurare le regole per specificare gli indirizzi e-mail in base alla quota utente.
