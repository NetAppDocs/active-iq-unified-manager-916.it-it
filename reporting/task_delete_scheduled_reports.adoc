= Elimina i report pianificati
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Dopo aver pianificato i report, è possibile eliminarli dalla pagina Pianificazioni report.

.Prima di iniziare
* È necessario disporre del ruolo di Amministratore dell'applicazione o Amministratore dell'archiviazione.


.Passi
. Nel riquadro di navigazione a sinistra, fare clic su *Gestione archiviazione* > *Pianificazioni report*.
+
image::../media/scheduled_reports_2.gif[Uno screenshot dell'interfaccia utente che mostra i report pianificati.]

+
[NOTE]
====
Se si dispone delle autorizzazioni appropriate, è possibile rimuovere qualsiasi report e la relativa pianificazione dal sistema.

====
. Clicca sull'icona "Altro"image:../media/more_icon.gif[""] per la pianificazione che vuoi rimuovere.
. Fare clic su *Elimina*.
. Conferma la tua decisione.
+
Il report pianificato viene rimosso dall'elenco e non verrà più generato e distribuito secondo la pianificazione impostata.

+
[NOTE]
====
Se si elimina una visualizzazione personalizzata dalla pagina dell'inventario, verranno eliminati anche tutti i file Excel personalizzati o i report pianificati che utilizzano tale visualizzazione.

====
