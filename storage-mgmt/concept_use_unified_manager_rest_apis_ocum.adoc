= Utilizzare le API REST di Unified Manager
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Active IQ Unified Manager fornisce API REST per visualizzare le informazioni sul monitoraggio e la gestione dell'ambiente di archiviazione.  Le API consentono inoltre di effettuare il provisioning e gestire oggetti di archiviazione in base a criteri.

È inoltre possibile eseguire le API ONTAP su tutti i cluster gestiti ONTAP utilizzando il gateway API supportato da Unified Manager.

Per informazioni sulle API REST di Unified Manager, vederelink:../api-automation/concept_get_started_with_um_apis.html["Introduzione alle API REST Active IQ Unified Manager"] .
