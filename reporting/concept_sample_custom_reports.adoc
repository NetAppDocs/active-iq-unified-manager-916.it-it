= Esempi di report personalizzati
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Questi report personalizzati di esempio vengono comunemente utilizzati per aiutarti a identificare potenziali problemi e a rispondere a potenziali problemi prima che si verifichino.

L'elenco dei report in questa sezione non è esaustivo e verrà ampliato nel tempo.  È possibile suggerire report personalizzati da aggiungere a questa sezione fornendo feedback sulla documentazione.

[NOTE]
====
Per gestire i report è necessario disporre del ruolo di Amministratore dell'applicazione o Amministratore dell'archiviazione.

====
