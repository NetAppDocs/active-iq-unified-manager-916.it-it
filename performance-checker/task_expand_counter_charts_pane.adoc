= Espandi il riquadro Grafici contatore
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
È possibile espandere il riquadro Grafici contatore in modo che i grafici siano più grandi e leggibili.

Dopo aver definito gli oggetti di confronto e l'intervallo di tempo per i contatori, è possibile visualizzare un riquadro Grafici contatori più grande.  Per espandere il riquadro, utilizzare il pulsante *<* al centro della finestra Performance Explorer.

.Fare un passo
. Espandi o riduci il riquadro *Grafici dei contatori*.


|===
| A... | Fai questo... 


 a| 
Espandi il riquadro Grafici contatore per adattarlo alla larghezza della pagina
 a| 
Fare clic sul pulsante *<*



 a| 
Ridurre il riquadro Grafici contatore nella metà destra della pagina
 a| 
Fare clic sul pulsante *>*

|===
