= Esempi di report di Microsoft Excel
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Questi report di esempio di Microsoft Excel hanno lo scopo di presentare le opzioni di reporting disponibili utilizzando le funzionalità avanzate di Excel.

Le funzionalità avanzate di Excel consentono di creare un'ampia gamma di report specifici per le tue esigenze.  Per informazioni complete su come utilizzare Excel, consultare la documentazione del prodotto.

[NOTE]
====
Per gestire i report è necessario disporre del ruolo di Amministratore dell'applicazione o Amministratore dell'archiviazione.

====
