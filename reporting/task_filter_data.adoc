= Filtra i dati
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Filtra i dati per assicurarti che i risultati corrispondano ai requisiti del tuo report.  Il filtraggio consente di visualizzare solo i dati di proprio interesse.

.Passi
. Fare clic sull'icona del filtro per aggiungere filtri e focalizzare i risultati che si desidera visualizzare, quindi fare clic su *Applica filtro*.
+
image::../media/filter_cold_data_2.png[Uno screenshot dell'interfaccia utente che mostra come aggiungere e applicare filtri.]

. Assegna un nome alla vista non salvata per salvare le modifiche.
