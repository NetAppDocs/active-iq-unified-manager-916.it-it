= Tipiche disposizioni delle finestre
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Conoscere i layout tipici delle finestre aiuta a navigare e utilizzare Active IQ Unified Manager in modo efficace.  La maggior parte delle finestre di Unified Manager sono simili a uno dei due layout generali: elenco oggetti o dettagli.  L'impostazione di visualizzazione consigliata è almeno 1280 per 1024 pixel.

Non tutte le finestre contengono tutti gli elementi nei diagrammi seguenti.
