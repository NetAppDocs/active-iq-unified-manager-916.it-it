= Backup e ripristino tramite NetApp Snapshots
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Una copia Snapshot NetApp crea un'immagine puntuale del database e dei file di configurazione di Unified Manager che è possibile utilizzare per il ripristino in caso di errore del sistema o perdita di dati.  È possibile pianificare periodicamente la scrittura di una copia Snapshot su un volume di uno dei cluster ONTAP , in modo da avere sempre una copia aggiornata.

[NOTE]
====
Questa funzionalità non è disponibile per Active IQ Unified Manager installato su un'appliance virtuale.
