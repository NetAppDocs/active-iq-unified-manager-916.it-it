= Gestire le pianificazioni dei report
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Puoi gestire le pianificazioni dei tuoi report dalla pagina Pianificazioni dei report.  È possibile visualizzare, modificare o eliminare le pianificazioni esistenti.

[NOTE]
.Prima di iniziare
====
Non è possibile pianificare nuovi report dalla pagina Pianificazioni report.  È possibile aggiungere report pianificati solo dalle pagine dell'inventario degli oggetti.

====
* È necessario disporre del ruolo di Amministratore dell'applicazione o Amministratore dell'archiviazione.


.Passi
. Nel riquadro di navigazione a sinistra, fare clic su *Gestione archiviazione* > *Pianificazioni report*.
. Nella pagina Pianificazioni dei report:
+
[cols="2*"]
|===
| Se lo desidera... | Poi... 


 a| 
Visualizza una pianificazione esistente
 a| 
Scorrere l'elenco dei report esistenti utilizzando le barre di scorrimento e i controlli della pagina.



 a| 
Modifica una pianificazione esistente
 a| 
.. Clicca sull'icona "Altro"image:../media/more_icon.gif[""] per la pianificazione che desideri utilizzare.
.. Fare clic su *Modifica*.
.. Apportare le modifiche necessarie.
.. Fare clic sul segno di spunta per salvare le modifiche.




 a| 
Elimina una pianificazione esistente
 a| 
.. Clicca sull'icona "Altro"image:../media/more_icon.gif[""] per la pianificazione che desideri utilizzare.
.. Fare clic su *Elimina*.
.. Conferma la tua decisione.


|===
