= Che cosa è RBAC
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
RBAC (controllo degli accessi basato sui ruoli) offre la possibilità di controllare chi ha accesso a varie funzionalità e risorse nel server Active IQ Unified Manager .
